De asemenea, CSS este unul din exemplele limbajelor computerizate care nu pot fi numite de programare. Lui CSS îi lipsesc elementele de limbaj diferite, foarte semnificative, ce permit limbajelor de programare să formuleze logica programării. Logica de programare permite ca procesul de rezolvare a unei probleme din viața reală să fie compilat în codul funcțional al programului.
