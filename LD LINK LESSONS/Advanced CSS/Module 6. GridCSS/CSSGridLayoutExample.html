Article Title

În felul acesta a fost creat un CSS Grid cu cinci elemente. Trei coloane au fost create explicit, folosind proprietatea grid-template-columns, în timp ce crearea rândului a fost lăsată pentru CSS Grid. Având în vedere că grid-ul are cinci elemente, într-o astfel de situație se va face crearea automată a două rânduri implicite în cadrul grid-ului. Astfel de rânduri create implicit pot fi influenţate utilizând proprietatea grid-auto-rows. Valoarea acestei proprietăţi este definită astfel încât rândurile create automat să aibă o înălțime minimă de 100px, în timp ce înălțimea maximă va depinde de înălțimea conținutului. CSS Grid-ul tocmai descris este ilustrat în imaginea 22.3.
