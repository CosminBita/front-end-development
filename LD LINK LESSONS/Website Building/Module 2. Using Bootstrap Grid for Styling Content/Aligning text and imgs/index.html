In modulul anterior am prezentat elementele de bază ale utilizării cadrului software Bootstrap pentru dezvoltarea front-end. Pe lângă demonstrarea diferitor abordări ale integrării Bootstrap, modulul anterior se ocupa în principal de funcţionalitatea de bază Bootstrap - grila Bootstrap. Cu toate acestea, deşi pentru mulţi prima asociere cu Bootstrap este un sistem de creare a layoutului foarte puternic, Bootstrap este mult mai mult decât atât. Are un set imens de stilizări pentru orice element care poate fi găsit într-un document HTML. Astfel, Bootstrap permite ca aspectul şi comportamentul conţinutului complet al site-ului să fie adaptate la cadrul de proiectare Bootstrap şi, astfel, să obţină un aspect uniform atunci când se utilizează site-uri cu Bootstrap în fundal.
